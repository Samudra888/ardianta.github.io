19 Dec 2017

Penasaran dengan ekstensi ini. Lansung saja saya coba install.

Pada deskripsinya menjelaskan…

…dengan ekstensi ini, kita bisa:

  1. Buat web hugo baru
  2. Buat artikel
  3. Build site
  4. Menajalankan server dan mematikannya
  5. Download Tema

Ekstensi Hugofy ini bisa kita gunakan dengan menekan tombol Ctrl+Shift+p lalu mengetik hugofy dan memilih perintah yang ingin dijalankan.

Tapi begitu saya coba menjalankan server, terjadi error seperti ini:

Sebagian dari perintah-perintah Hugofy sudah biasa saya ketik lewat terminal.

Jadi saya memutuskan untuk uninstall saja, karena lebih cepat rasanya lewat terminal.

Mungkin buat yang belum terbiasa menggunakan terminal bisa mencobanya.

Kalau saya sih lebih suka pakai terminal.
